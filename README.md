# HR Analytics Dashboard (Power BI)

## Overview
This project analyzes employee attrition and workforce demographics using Power BI.
It provides insights into retention, satisfaction, and performance trends.

## Key Metrics
- Total Employees: 1470
- Attrition Cases: 237
- Attrition Rate: 16.1%
- Workforce Average Age: 37
- Average Salary: 6.5K
- Average Tenure: 7 years

## Dashboards
- Attrition by Gender, Age, Salary, Education Field
- Attrition by Years at Company
- Job Satisfaction Matrix by Role
- Performance Rating Trends

## Business Impact
- Identifies high‑risk groups for attrition
- Highlights salary and tenure patterns linked to exits
- Supports HR in designing retention strategies
